
ProgEdu[7]là một hệ thống đánh giá chất lượng code dành cho các giảng viên và sinh viên được phát triển bởi nhóm nghiên cứu của GS H.-M. Chen tại đại học Feng-Chia, Đài Loan. ProgEdu cung cấp các công cụ để đo lường chất lượng code, phân tích và cải thiện kỹ năng lập trình của người dùng và được cung cấp như một phần mềm nguồn mở.
